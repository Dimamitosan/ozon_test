# ozon_test

ссылка на страницу - https://dimamitosan.github.io/ozon_test/progress.html

после ввода значения, для запуска анимации, необходимо нажать Enter на клавиатуре

доступ к состоянию происходит через консольные команды 

progressBlock.setValue(50) - установка значения прогресса 
progressBlock.setHidden(true) - установка значения видимости
progressBlock.setRotation(false) - установка значения вращения 

progressBlock.getValue() - получение значения прогресса
progressBlock.getHidden() - получение значения видимости
progressBlock.getRotation() - получение значения вращения
